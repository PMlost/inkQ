Privacy Policy

Your privacy is important to us

Effective Date: August 12, 2025 Last Updated: August 12, 2025

Introduction

featherScroll ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application and services integrated with the Pi Network ecosystem.

Information We Collect

Personal Information

  • Pi Network Integration: Pi username, Pi wallet address (when you connect)
  • Account Data: Email address (if provided), user preferences
  • Usage Data: App interactions, quote preferences, daily engagement metrics

Automatically Collected Information

  • Device Information: Device type, operating system, unique device identifiers
  • Analytics Data: App usage patterns, feature interactions, session duration
  • Location Data: General location (country/region) for content localization

Pi Network Specific Data

  • Pi Authentication: Pi Network user ID and public profile information
  • Transaction Data: Pi payment history (if applicable for premium features)
  • Blockchain Data: Public Pi Network transaction records

How We Use Your Information

Core App Functions

  • Deliver daily inspirational quotes and content
  • Personalize your experience based on preferences
  • Maintain app functionality and user accounts

Pi Network Integration

  • Process Pi Network authentication and payments
  • Verify user identity through Pi Network protocols
  • Enable Pi-based features and transactions

Information Sharing and Disclosure

We DO NOT Share

  • Personal identification information with third parties
  • Private messages or personal preferences
  • Financial information beyond Pi Network transactions

We MAY Share

  • Pi Network: Authentication data as required by Pi Network protocols
  • Analytics Services: Anonymized usage data for app improvement
  • Legal Requirements: Information when required by law or to protect rights

Data Security

We implement industry-standard security measures:

  • Encryption: All data transmitted is encrypted using SSL/TLS
  • Pi Network Security: Follow Pi Network's security best practices
  • Access Controls: Limited access to personal data on need-to-know basis
  • Regular Audits: Periodic security assessments and updates

Your Rights and Choices

Account Control

  • Access: View your personal information
  • Update: Modify account preferences and settings
  • Delete: Request account deletion (subject to Pi Network requirements)

Contact Information

For privacy-related questions or concerns:

  • Email: privacy@featherscroll.com
  • Pi Network: Contact through Pi Network messaging
  • Support: In-app support feature

By using featherScroll, you acknowledge that you have read and understood this Privacy Policy.
